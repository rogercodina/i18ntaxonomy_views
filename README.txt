Taxonomy localization for views
===============================

If you have localized taxonomy terms and your terms appears in default language
(in your views) even if you have selected another language, this module is for
you!

This module is based on this post:
http://ggordan.com/post/localising-taxonomy-terms-in-views.html

This module was downloaded from https://github.com/rogercodina/i18ntaxonomy_views
